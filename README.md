# flix-talk

넷플릭스를 사용하여 “네이버 실시간 톡” 처럼 프로그램 별로 사용자 간 대화가 가능하면 좋겠다는 아이디어에서 시작하여, 카카오톡의 주요 기능을 결합한 Flix Talk을 구상했다.
<br>
<br>
[Login]<br>
이메일 또는 휴대폰 번호로 로그인 한다.

[Programs]<br>
카카오톡의 친구 목록처럼, 넷플릭스 프로그램들을 리스트 형태로 보여준다.

[Chats]<br>
선택한 프로그램별로 채팅방에 참여할 수 있는 목록들이 나타나며, 각 방에서는 프로그램 소개, 주요 등장인물 정보, 줄거리 등을 확인할 수 있다.

[Find]<br>
드라마, 영화, 게임 카테고리를 쉽게 탐색할 수 있도록 UI를 구성했으며, 카카오톡 오픈채팅 기능을 활용해 프로그램에 대해 다른 사용자와 대화를 나눌 수 있는 기능을 추가했다.

[More]<br>
사용자 프로필과 함께, 사용자가 저장한 프로그램이나 곧 공개될 콘텐츠를 확인할 수 있는 섹션을 만들었다.
